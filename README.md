## Veiculo

Herança da classe Veiculo com carro e moto.
Polimorfismo na classe moto

## Conta com cliente

Um Banco aonde é possivel cadastrar o cliente e criar uma conta.
O cliente pode efetuar as seguintes operações

* ----------1 - Nova conta--------------------
* ----------2 - Depositar na conta------------
* ----------3 - Retirar da conta--------------
* ----------4 - Alterar conta-----------------
* ----------5 - Excluir conta-----------------
* ----------6 - Ver conta---------------------
* ----------0 - Sair--------------------------
